## 📐 Conception & Présentation d'Otto-MKS
Le robot Otto original est un robot humanoïde open-source, modulaire et programmable, largement conçu pour l'impression 3D.

Sur les bases de ce projet mondial, le MakerSpace d'UniLaSalle Amiens a développé une version sur mesure : **l'Otto-MKS**. Cette version améliorée intègre une connectivité sans fil avancée (Bluetooth et WiFi) permettant de piloter le robot directement depuis un smartphone, ouvrant ainsi de nouvelles perspectives de contrôle et d'interaction pour les épreuves.
